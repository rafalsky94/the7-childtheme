Description
-------------------

Panoply is a free one page multi-purpose WordPress theme. It's perfect for web agency business,corporate business, personal and parallax business portfolio, photography sites and freelancer. Theme is built on BootStrap with parallax support, is responsive, clean, modern, flat and minimal. 

Panoply also supports WP Customizer, Custom Menu, Custom Background and Custom Header, enabling flexible configuration.
